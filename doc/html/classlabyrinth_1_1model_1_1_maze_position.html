◆ getDirectionTo()

MazeDirection labyrinth::model::MazePosition::getDirectionTo ( const MazePosition &  neighbor) const

Gets the direction to the given position.

For instance, the current position (0; 0) and (0; 1) is given. The direction is right because (0; 1) is on the right of (0; 0).

The given position should be a direct neighbor. A direct neighbor is right next to this position.

The given position should be different than this one.

Parameters
